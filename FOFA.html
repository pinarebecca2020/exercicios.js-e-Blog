O que é HTML?




HTML (Hypertext Markup Language) é uma linguagem de marcação.

É responsável pela estruturação de uma página web. A marquação é feita por meios de etiquetas, chamadas comumente de tags.

No HTML 5 utilizamos a tag <!DOCTYPE html> para informar o navegador sobre o tipo de documento esperado, no caso HTML.

  • A tag <html lang="language"> serve para especificar qual a linguagem da página, facilita na hora de traduzir a página para outra linguagem. Obs: lang= é um atributo.
  • A tag <title> serve para colocar o título (nome que aparece na aba) da página.
  • A tag <h1...h6> representa os níveis de títulos e suas importâncias(tamanhos). Onde o h1 é o mais importante, logo de maior tamanho e o h6 de menor importância e tamanho.
  • A tag <p> é utilizada para especificar um parágrafo.
  • A tag <head> é como um cabeçalho da página, onde contém todas as informações passadas para o navegador, é nela que você inclui o título, especifica a linguagem, o dicionário, inclui um arquivo externo de css, etc.
  • A tag <body> é como o corpo da página, onde você insere as informações que serão exibidas na página.
  • A tag <meta charset="dicionário"> serve para passar informações para o navegador. O atributo charset é utilizado para especificar um conjunto de caracteres; dicionário.
  • A tag <strong> serve para colocar em negrito.
  • A tag <em> serve para colocar em itálico.

    • E o CSS?

    CSS (Cascading Style Sheets), em outras palavras, é uma linguagem de estilo em cascata.
